# Triage labels

The engineering skills use these exact label names:

| Canonical role | Repository label | Meaning |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | Maintainer evaluation is needed |
| `needs-info` | `needs-info` | Waiting for the reporter |
| `ready-for-agent` | `ready-for-agent` | Fully specified and safe for an unattended agent |
| `ready-for-human` | `ready-for-human` | Human implementation is needed |
| `wontfix` | `wontfix` | The issue will not be actioned |

Do not create or rename remote labels merely by reading this file. A task needs explicit GitHub mutation authority before changing labels.
